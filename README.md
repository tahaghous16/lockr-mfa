# Lockr 🔐

Lockr is a modern, secure Multi-Factor Authentication (MFA) application built using the MERN stack (MongoDB, Express, React, Node.js). 

It is designed to provide an extra layer of security beyond traditional passwords by requiring users to verify their identity using a secondary method, such as a Time-based One-Time Password (TOTP) from an authenticator app.

## Key Features

- **Secure Authentication**: Traditional email/password login fortified with MFA.
- **Two-Factor Setup (2FA)**: Easy enrollment process for users to set up authentication apps (like Google Authenticator or Authy).
- **Modern UI**: A responsive, beautiful, and user-friendly interface built with React and Tailwind CSS.
- **JWT Authorization**: Secure, stateless session management using JSON Web Tokens.

## Tech Stack

- **Frontend**: React.js (Vite), Tailwind CSS, React Router DOM, Axios
- **Backend**: Node.js, Express.js, MongoDB (Mongoose), JSON Web Tokens (JWT)
